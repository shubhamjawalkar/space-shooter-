# 🚀 Space Shooter Game (Python + Pygame)

A fast-paced 2D space shooter built using Python and Pygame. Control your spaceship, shoot enemies, and survive as long as possible.

---

## 🎮 Features

* 🚀 Smooth player movement
* 🔫 Shooting mechanics
* 👾 Random enemy spawning
* 💥 Collision detection
* 🧠 Score tracking
* ⚡ 60 FPS optimized gameplay

## ⚙️ Installation

```bash
git clone https://github.com/shubhamjawalkar/space-shooter.git
cd space-shooter
pip install -r requirements.txt
python src/main.py
```

---

## 🎮 Controls

* ⬅️ Move Left
* ➡️ Move Right
* SPACE → Shoot
